## Name
**Cove.**

A cove is a small, sheltered inlet — a private, calm, protected place that is *yours*. It captures both
halves of the product: **privacy / shelter** (your data stays in your cove; nothing leaves) and the
[calm-by-design](../features/07-calm-design.md) aesthetic. It's short, easy to say and spell, and works
as an umbrella over many tools.

- **Product / brand name:** Cove
- **Category descriptor:** "a local-first browser tool" (use after the name on first mention)
- **Repository folder:** `cove` (matches the product name)

## Tagline
**"Real tools. Nothing leaves your device."**
Short form: *"Your files never leave."*

## Tool naming
Tools are **"Cove &lt;Verb/Noun&gt;"**, kept plain and descriptive:

| Tool | Brand name | Status |
| ---- | ---------- | ------ |
| Video compressor (v1 flagship) | **Cove Compress** | MVP |
| PDF redactor | **Cove Redact** | Phase 3 |
| CSV ↔ JSON ↔ Parquet converter | **Cove Convert** | Phase 3 |
| Background remover | **Cove Cutout** | Phase 4 |

## Voice & tone
Calm, plain, honest. We explain rather than hype; we never use fear, urgency, or dark patterns
([Calm by Design](../features/07-calm-design.md)). Prefer concrete claims ("nothing is uploaded") over
adjectives ("ultra-secure").

## Usage
- Capitalize as **Cove**. Don't stylize (no "COVE", no "cove.").
- First mention on a page/doc: "**Cove** — a local-first browser tool — …", then "Cove" thereafter.
